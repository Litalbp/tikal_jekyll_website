Tikal's Fuse day 25/12/12

by
tags:
25
Nov. 2012

Tikal's next fuse day will be on Tuesday 25/12/12 @ Tikal's offices. 

The kickoff will be at 24/12/12, after Tikal's  status meeting. During the kickoff the session leaders will give a short presentation on the session subjects.

 


The sessions are (click on the title to register to the session) :

  1. ALM - Pallet (DevOps Provisioning) or Automation of Development/Runtime workstation (Vagrant) By: Itai Or.
  2. Java - Polyglot persistence using spring data by : Yanai & Andrew.
         In this Fuse day we'll have a hands-on session on Spring Data - A Spring umbrella open source project which contains many
    sub-projects that are specific to a given data store (not necessary RDBMS)  like Redis, MongoDB, Neo4j, Hadoop and others.
  3. Javascript - taking the Mozilla HTML5/JS API Challenge by: OrenF. 

    We'll be taking one of more of mozilla's dev derby challenges.

    This session will give you the opportunity to explore and enjoy the bleeding edge features

    of HTML5 JS API's, such as: Camera API, GEO location, Audio, webgl, 3D CSS etc...

    Session members can pick any idea to implement using various api's to create any original POC.

  4. .NET - Developing on Windows 8 - IgorZ & Menny.
    Get your hands dirty with windows 8 application. Learn about WinRT’s life cycle, API, Tiles and basic features. See if you can build a ‘comics like’ application using JavaScript over WinRT, debug it and maybe even deploy to the windows store.
  5. Mobile - Developing Tikal Time Tracker using Phonegap - DovG.
     Investigate and implement JavaScript Mobile Frameworks.  Optimize the existing Tikal TIme Tracker App.  Improve the existing app and allow it to communicate directly with Tikal's Web Time Tracker Portal.
  6. Python/Django - Mehameme - Israel meme generator by: Alex Arshavski
     The project will be called "Mehameme". Its going to be a web site like http://memegenerator.net/ but in hebrew.
    The technologies we gonna use are:
    1.) Python / Django
    2.) Google App Engine
    3.) Google Cloud SQL
    4.) Git / Github
    Recommended reading before the fuze day:
    1.) Django tutorial: https://docs.djangoproject.com/en/1.4/intro/tutorial01/
    2.) Google app engine tutorial: https://developers.google.com/appengine/docs/python/gettingstartedpython27/
    3.) Google cloud sql: https://developers.google.com/cloud-sql/docs/before_you_begin
    4.) Git: http://git-scm.com/book/en/Git-Basics

 

Registration to each one of the Fuse day session, on Tikal's Meetup.
